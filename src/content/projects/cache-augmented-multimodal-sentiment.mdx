The system treats sentiment as a multimodal inference problem. Instead of forcing text, audio, and visual expression into separate demos, the project brings them into one Streamlit interface and uses embeddings to make repeated prediction work more efficient.

The publication is available at [IEEE Xplore](https://ieeexplore.ieee.org/document/11414590).
